# Free open-source MCP catalog (Aurora)

Total: **50** (cap 50)

Local-first, no paid API keys required for core use.


## Browser & automation

- **Playwright Browser** (`playwright_ms`) — Microsoft Playwright MCP — browse, click, screenshot, scrape (local).
- **Playwright (ExecuteAutomation)** (`playwright_executeautomation`) — Popular Playwright automation MCP for testing/scraping.
- **Puppeteer** (`puppeteer_official`) — Official Puppeteer browser automation MCP (local Chrome).
- **Browser Tools** (`browser_tools`) — Browser tools MCP for audits/console/network style workflows.
- **Playwright (AutomataLabs)** (`playwright_automatalabs`) — Alternative Playwright MCP server.
- **Screenshot** (`screenshot_mcp`) — Capture screenshots via MCP.

## Code & reasoning

- **Aurora Code Tools** (`aurora_code`) — Python outline/syntax check, TODO scan, LOC, markdown TOC.
- **Sequential Thinking (official)** (`mcp_sequential_thinking`) — Structured multi-step reasoning tool — great for hard problems.

## Core local (always free)

- **Aurora Demo** (`aurora_demo`) — Built-in demo tools: echo, reverse, time, add (always free).
- **Aurora Utils** (`aurora_utils`) — Hash, base64, JSON, regex, diff, slugify, safe calculator — offline.
- **Time (Python official)** (`mcp_time_py`) — Timezone-aware time tools via official Python MCP server.

## Data & storage

- **Aurora Knowledge DB** (`aurora_knowledge`) — Local SQLite notes, todos, key-value memory — no cloud.
- **Memory Graph (official)** (`mcp_memory`) — Official knowledge-graph memory server (local).
- **SQLite (Python official)** (`mcp_sqlite_py`) — Query/inspect a local SQLite database file.
- **SQLite (npm)** (`sqlite_npm`) — SQLite MCP server (Node).
- **Postgres (local)** (`postgres_local`) — Official Postgres MCP — free with local Postgres URL.
- **Redis (local)** (`redis_local`) — Official Redis MCP — free with local Redis (no Upstash required).
- **SQLite server pkg** (`sqlite_mcp_server_pkg`) — mcp-server-sqlite npm package.

## Developer utilities

- **Calculator** (`calculator_mcp`) — Precise calculator MCP.
- **MCP Installer** (`mcp_installer`) — Install/manage other MCP servers from chat (Anais Betts).
- **Calculator (WrtnLabs)** (`calculator_wrtnlabs`) — Alternative calculator MCP.
- **Everything (protocol lab)** (`mcp_everything`) — Official protocol exercise server (tools/resources/prompts demos).
- **HuggingFace Spaces** (`hfspace`) — Call public HF Spaces tools (free public endpoints; community limits possible).
- **MCP Inspector** (`mcp_inspector`) — Official inspector/debug companion for MCP servers.
- **Smithery CLI** (`smithery_cli`) — Discover/run community MCP servers via Smithery CLI.

## Docs & content

- **Context7 Docs** (`context7`) — Up-to-date library docs lookup for coding agents (free tier community MCP).
- **YouTube Transcript** (`youtube_transcript`) — Fetch YouTube transcripts locally (no paid API).
- **MarkItDown** (`markitdown_cli`) — Convert PDF/Office/HTML to markdown via Microsoft markitdown (local).
- **YouTube Data** (`youtube_data`) — YouTube data helpers MCP.

## Files & workspace

- **Aurora Workspace FS** (`aurora_workspace`) — Sandboxed read/write/list/search/tree for Aurora workspace.
- **Filesystem (workspace only)** (`mcp_filesystem_workspace`) — Official filesystem MCP scoped only to Aurora workspace.
- **Filesystem (home)** (`mcp_filesystem_home`) — Official filesystem MCP scoped to /home/user.
- **Filesystem (official)** (`mcp_filesystem`) — Official MCP filesystem server for sandboxed directories.
- **Filesystem Advanced** (`fs_cyanheads`) — Extended filesystem operations MCP.

## Git & version control

- **Git (Python official)** (`mcp_git_py`) — Read git status/log/diff and more for a local repo.
- **Git Advanced** (`git_cyanheads`) — Feature-rich local git MCP (branches, commits, diffs).
- **Git (npm simple)** (`git_npm_simple`) — Simple git MCP package.
- **Git server pkg** (`git_mcp_server_pkg`) — mcp-server-git npm package (local).

## System / shell / desktop

- **Desktop Commander** (`desktop_commander`) — Powerful local desktop/terminal/file orchestration MCP.
- **Docker (Python)** (`docker_mcp_py`) — Manage local Docker via MCP (needs Docker daemon).
- **Shell Commands** (`mcp_commands`) — Run allowlisted shell commands via MCP.
- **Docker (npm)** (`docker_mcp_npm`) — Docker MCP via npm package.
- **Docker MCP alt** (`docker_mcp_alt`) — Alternate docker MCP package name.
- **Shell (Python)** (`mcp_server_shell_py`) — Python shell MCP server.
- **Shell MCP** (`shell_mcp`) — Lightweight shell execution MCP.
- **Webcam** (`webcam_mcp`) — Local webcam capture MCP (if hardware available).
- **macOS Automator** (`macos_automator`) — macOS automation MCP (only useful on macOS hosts).

## Web (no API key)

- **Aurora Free Web** (`aurora_web_free`) — DuckDuckGo search + HTTP fetch without API keys.
- **Fetch (Python official)** (`mcp_fetch_py`) — Fetch and extract web content (local server, free).
- **Fetch (npm simple)** (`fetch_npm_simple`) — Simple fetch MCP package.
